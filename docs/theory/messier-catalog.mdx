# 梅西耶星表

梅西耶星表（Messier Catalog）是法国天文学家查尔斯·梅西耶（Charles Messier）于 18 世纪编制的天体列表，最初目的是标记天空中容易被误认为彗星的"模糊天体"。如今它已成为天文爱好者最常使用的深空天体观测清单，收录了 110 个星系、星云和星团[^1]。

:::info[梅西耶天体类型色标]

- 🟦 蓝色背景 — 疏散星团
- 🟨 黄色背景 — 球状星团
- 🟥 粉色背景 — 发射星云
- 🟩 绿色背景 — 行星状星云
- 🟪 紫色背景 — 超新星遗迹
- 🟡 浅黄背景 — 星系
- ⬜ 白色背景 — 其他

:::

## 梅西耶马拉松

每年三月至四月间，当太阳位于没有梅西耶天体的天区时，观测者有机会在一夜之内看到全部 110 个梅西耶天体，这被称为**梅西耶马拉松**（Messier Marathon）。

广州的纬度（约 23°N）适合观测大部分梅西耶天体，但部分偏南天体（如 M7、M54、M55、M69、M70 等）需要良好的南方视野。北天最亮的几个梅西耶天体（M31、M42、M45）即使是城市光害条件下也能轻松看到，是新手入门的最佳目标。
