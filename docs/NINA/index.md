## 关于 N.I.N.A.

欢迎来到 N.I.N.A. — Nighttime Imaging 'N' Astronomy（夜间拍摄与天文学）。如果你正在阅读本文档，你很可能已经[下载](//nighttime-imaging.eu/download/)了这个软件——恭喜你！

N.I.N.A. 专为利用序列模型进行自动化深空天体（DSO）拍摄而设计。无论你是 DSO 拍摄的新手还是经验丰富的老手，N.I.N.A. 的目标都是让拍摄过程更轻松、更快捷、更舒适。N.I.N.A. 致力于让复杂的概念和操作变得通俗易懂、便于执行，让你能把更多时间花在拍摄上，而不是操心设备上。

N.I.N.A. 项目同时也是一个[自由开源软件（FOSS）](//en.wikipedia.org/wiki/Free_and_open-source_software)应用，依据 [Mozilla Public License Version 2.0](//www.mozilla.org/en-US/MPL/2.0/) 许可协议进行授权和分发。其维护、成熟发展和进一步开发由来自世界各地的志愿者团队完成。如果你有兴趣帮助该项目或为其做出贡献，欢迎加入项目的 Discord 服务器（见下文）并阅读[贡献者指南](//github.com/isbeorn/nina/blob/develop/CONTRIBUTING.mdd)。
