# 设置星图软件

## Stellarium

要使 Stellarium 能够同步坐标到 N.I.N.A.，需要启用远程控制插件并启动服务器。
首先在 Stellarium 中按 F2 进入配置。
![配置](../images/troubleshooting/planetarium/stellarium_configuration.png)

进入插件标签页，搜索"Remote Control"插件并选中它。
在右侧，勾选"启动时加载"选项。

![插件](../images/troubleshooting/planetarium/stellarium_plugins.png)

插件启用后，还需要启用插件服务器。为此，点击"配置"按钮，勾选"启用服务器"以及"启动时自动启用"，以免每次都要手动操作。

![远程控制](../images/troubleshooting/planetarium/stellarium_remote_control.png)
